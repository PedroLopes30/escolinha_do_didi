# Escolinha do Didi
## Description
This application was developed to manage student enrollments. The client creates a user account and registers the student (for which he is responsible) they wish to enroll in Escolinha do Didi; the manager (admin) reviews and approves the enrollment. It was created to optimize the enrollment process at the tutoring school.
